📅 Slotify Technical Architecture

Current State: Slotify has a modular structure but no formal plugin system yet.

Approach: Create add-ons as npm packages in the monorepo's /packages directory.

Integration Points: Service layer, middleware pipeline, API routes.

Tech Stack: React + Vite (frontend), Express.js + MongoDB (backend), pnpm workspaces.
